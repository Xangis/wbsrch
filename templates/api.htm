WbSrch API

WbSrch has an API available for retrieving search results or domain and keyword details. It can be used to build your own search (or metasearch) engine, or to use the data as part of an SEO or keyword tool.

Please note that our policy does not allow using WbSrch results in a meta-search engine unless you retieve them via the API.

Pricing

Pricing is $2 per 1000 query credits. Bulk pricing is available for high-volume users.

Terms

All API usage shall be pre-paid unless otherwise arranged. There is a minimum initial deposit of $10.

You may cache the results of a query and may store any non-search results indefinitely.

You may cache search results for up to 30 days. After that time, the data must be discarded unless it is being used for research. You may not display search results older than 30 days to end-users.

You may combine our API results with results from any other source. Use in a meta search engine is not only allowed, but encouraged.

Usage

All API requests must contain 'Authorization: Token <yourtoken%gt;' in the header.

Endpoints

Domain Link Rank

Cost: 1 credit

curl -X GET https://wbsrch.com/api/domain_link_rank/?domain=example.com -H 'Authorization: Token <yourtoken>'

Requires

  • domain: (String) The domain being queried.

Returns

{"domain":"example.com","wbrank":4,"excluded":false,"domains_linking_in":245,"excluded_reason":null}

Fields

  • domain: (String) The domain being queried.
  • wbrank: (Integer) The WbSrch WbRank (link rank) for that domain.
  • excluded: (Boolean) Whether the domain is excluded from the WbSrch index.
  • domains_linking_in: (Integer) The number of domains linking to the requested domain according to the WbSrch index.
  • excluded_reason: (String) The reason the domain has been excluded from the WbSrch index, if any.
