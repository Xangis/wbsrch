A Web Browser

WbBrowse is a full-featured web browser available for Windows, OSX and .deb-based Linux.

WbBrowse is a modern HTML5 browser based on the Chromium (Blink) web engine. It has a tabbed browsing expeience and supports bookmarking, view source, print, file downloads, private browsing, and web history.

According to html5test.com, WbBrowse scores 472 out of 555 points when running on Windows 10.
